Cabeçalhos

(Voltar)

Os cabeçalhos em HTML são usados para definir títulos e subtítulos de diferentes níveis de importância em uma página web.

  • h1: é usada para o título principal, que é o mais importante e geralmente aparece maior e em negrito.
  • h2: é usada para subtítulos e assim por diante, até h6.
  • h6: é o nível de cabeçalho menos importante e aparece menor.
No contexto geral, eles ajudam a estruturar o conteúdo da página e são essenciais para a acessibilidade e SEO.

Parágrafo

(Voltar)

A tag p define um parágrafo de texto. É um dos elementos mais básicos e amplamente utilizados em HTML para estruturar blocos de texto. O navegador adiciona automaticamente um espaço em branco (margem) antes e depois de cada parágrafo, tornando o texto mais legível.

Marca de Texto

(Voltar)

A tag mark é usada para destacar partes do texto que são de interesse ou que precisam de atenção. O texto dentro da tag mark geralmente aparece com um fundo amarelo, similar ao uso de um marcador de texto físico em papel.

Texto Pequeno

(Voltar)

A tag small faz o texto parecer menor do que o texto circundante. É frequentemente usada para notas de rodapé, avisos ou qualquer outro texto que deve ser mostrado em um tamanho de fonte reduzido.

Texto Itálico

(Voltar)

A tag i estiliza o texto em itálico, geralmente usada para enfatizar uma palavra ou frase, títulos de livros, termos técnicos, estrangeirismos ou pensamentos de personagens em uma narrativa.

Texto Sublinhado

(Voltar)

A tag u sublinha o texto. Embora seja usada com menos frequência devido a questões de usabilidade e acessibilidade (pois os links geralmente são sublinhados), ainda pode ser útil para enfatizar texto em algumas situações.

Texto em Negrito

(Voltar)

A tag strong indica que o texto é importante, semântica e visualmente. O texto dentro da tag strong geralmente é renderizado em negrito pelos navegadores, indicando uma ênfase forte.

Lista Ordenada

(Voltar)

A tag ol cria uma lista ordenada, onde cada item é numerado. Esta tag é útil quando a ordem dos itens é importante, como em uma receita, instruções passo a passo ou uma lista de prioridades.

Lista não Ordenada

(Voltar)

A tag ul cria uma lista não ordenada, onde cada item é marcado com um ponto (bullet). É usada quando a ordem dos itens não é importante, como em listas de compras, características de produtos ou tópicos de discussão.

Item de Lista

(Voltar)

A tag li define um item em uma lista ordenada (ol) ou não ordenada (ul). Cada item de lista é representado por li, que pode conter texto, links, ou outros elementos HTML.

Link

(Voltar)

A tag a cria um hyperlink, que é um link clicável que leva o usuário a outra página ou seção da mesma página. O atributo href define o destino do link. Links são fundamentais para a navegação na web.

Subscrito

(Voltar)

A tag sub faz com que o texto apareça como subscrito, ou seja, um pouco abaixo da linha de texto normal. É frequentemente usada em fórmulas químicas ou matemáticas, como H₂O.

Sobrescrito

(Voltar)

A tag sup faz com que o texto apareça como sobrescrito, ou seja, um pouco acima da linha de texto normal. É usada em referências, exposições e ordens como em x².

Citação

(Voltar)

A tag blockquote define uma seção de texto que é uma citação de outra fonte. É geralmente exibida com recuo e pode incluir citações longas, discursos ou trechos de artigos.

Fonte

(Voltar)

A tag font era usada para definir a fonte, cor e tamanho do texto, mas é considerada obsoleta em HTML5. Atualmente, a estilização deve ser feita usando CSS para manter a separação de conteúdo e apresentação.

Texto Removido

(Voltar)

A tag del mostra texto que foi excluído de um documento. Visualmente, é exibido com uma linha através dele, indicando que foi removido ou está obsoleto.

Linha Horizontal

(Voltar)

A tag hr insere uma linha horizontal na página, usada para separar seções de conteúdo. É uma maneira visual de dividir temas ou seções em uma página.

Abreviação

(Voltar)

A tag abbr define uma abreviação ou acrônimo. O atributo title pode ser usado para fornecer a versão completa ou descrição do termo abreviado, melhorando a acessibilidade e usabilidade.
